# まちの店長ストーリー

2Dドット絵の店経営ゲームです。  
最初は自宅ショップから営業を始め、稼いだお金で新しい店を建てます。

## 遊び方

1. `index.html` をブラウザで開く
2. 右上 `+` ボタンで `町の店長ストーリー` パネルを開く
3. まず区画1の自宅ショップで営業し、売上を作る
4. 店の前で `E` / `Enter` を押すと店内に入り、同じ操作で外に出る
5. 在庫が減ったら `仕入れ`、余裕が出たら `拡張`
6. 資金が増えたら区画2・区画3に新しい店を `建築`
7. 外から来た客が店内で商品を選び、レジで会計して帰る

## 操作

- 移動: `WASD` または `矢印キー`
- 入店/退店: `E` または `Enter`
- 店内会計: レジ前で `E` または `Enter`
- 経営操作: 右パネルのボタン (`建築` / `仕入れ` / `拡張`)

## 仕様

- 店タイプ: 雑貨店 / パン屋 / 本屋 / 花屋
- 顧客は在庫と店の魅力度に応じて来店
- 1営業日ごとに固定費が発生
