// turbo-all

1. すべての変更ファイルをインデックスに追加する
   `git add .`

2. 作業内容に基づいた分かりやすいコミットメッセージを作成してコミットする
   `git commit -m "[AI Auto-sync] タスクの内容をここに記述"`

3. GitHubのメインブランチへ変更をアップロード（プッシュ）する
   `git push origin main`

## 注意事項
- このワークフローは、タスク（コードの修正、新規機能の追加など）が完了した直後に、ユーザーの確認を待たずに実行してください。
- 実行後は「作業内容をGitHubへ自動保存（プッシュ）しました！」と報告してください。
- エラーが発生した場合は、速やかにユーザーに報告し、解決策を提示してください。
